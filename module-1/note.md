## 🐍 What is Python?
- A high-level, interpreted programming language
- Created by **Guido van Rossum** in 1991
- Name comes from "Monty Python's Flying Circus" — not the snake
- Designed for readability and simplicity
- One of the most popular languages in the world — used in web, data science, AI, automation and systems engineering

## ⚙️ Interpreted vs Compiled
- **Compiled** (e.g. C, C++) — source code converted to machine code before running. Fast but platform-specific.
- **Interpreted** (e.g. Python) — code is read and executed line by line at runtime. Slower but portable and easier to debug.
- Python uses an interpreter called **CPython** (the standard implementation)

## 🖨️ print() function
The most basic Python function — outputs text to the screen.
```python
print("Hello, World!")
print("My name is Wai")
print(42)
print(3.14)
```
